# luci-theme-butongwifi
openwrt luci主题

> * git clone到openwrt的源码目录，如openwrt-src/package/mypackage/luci-theme-butongwifi
> * make menuconfig，进入luci->themes，选择luci-theme-butongwifi，保存退出，make V=99编译。
